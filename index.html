How to Use the New Build Generator

  1. Create a build with Nautsbuilder
  2. Copy the embed code from Nautsbuilder
  3. Paste the embed code into the text box
  4. Press enter to generate a build file
  5. Copy-paste the XML into a new Awesomenauts XML file or click Download to save it to your computer
